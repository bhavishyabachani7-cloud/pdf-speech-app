# 🔥 AI Speech Tool

Convert:
- 📄 PDF to Speech
- ✍️ Text to Speech
- 🖼️ Image to Speech (local)

## Features
- 🎙️ Multiple voice styles
- 🎧 Audio player
- ⬇ Download audio
- 📱 Mobile friendly

## How to use
1. Upload file or enter text
2. Select voice style
3. Click convert

## Built with
- Python (Flask)
- gTTS
